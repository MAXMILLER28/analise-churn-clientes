# 📊 Análise de Cancelamento de Clientes (Churn)

Projeto desenvolvido para identificar os principais fatores que levam clientes a cancelarem serviços.

## 🚀 Tecnologias
- Python
- Pandas
- Plotly
- XGBoost

## 📈 Etapas
- Análise exploratória
- Visualização de dados
- Machine Learning
- Identificação de variáveis importantes

## 💡 Insights
- Clientes inativos possuem maior chance de cancelamento
- Menor número de transações aumenta o risco de churn
- Comportamento do cliente influencia diretamente
